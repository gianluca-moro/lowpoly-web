About

This tool can be used to convert images to Low-Poly art style.
I really like low-poly art and thus, I wanted to create a tool that I can use to automatically create low-poly images.

The converter is implemented in JavaScript and works as follows
  1. The image is converted to gray-scale
  2. We then blur the gray-scale image twice with different blur strengths and calculate the difference between those images for each pixel.
    This will give a higher difference value for pixels that are "more interesting", e.g., on object edges or where colours change, while giving lower difference values for pixels in areas that are more uniform.
    For the image blur, we are using stackblur-canvas (npm), which gives us an almost Gaussian blur with great performance.
  3. We then perform Rejection Sampling on these pixel differences to randomly select n (= number of triangulation points) pixels.
    Note that since we perform the rejection sampling on the pixel differences, pixels that are "more interesting" will be sampled with a higher probability. This ensures that we have more smaller triangles in more interesting areas while fewer but larger triangles in less interesting areas.
  4. We then use Delaunay Triangulation to triangulate the sampled pixels, i.e., we create a set of triangles that do not overlap.
    For the triangulation, we are using Delaunator (npm).
  5. Finally, we draw the triangles on the output image. To get the colour for each triangle, we calculate the midpoint of the triangle and pick the colour of the original image at that pixel.
The source code is available on GitHub (lowpoly-js). I have also implemented the algorithm in Python, which is also available on GitHub (lowpoly-python).
